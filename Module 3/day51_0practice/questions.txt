Mini Task 1: Smart Home Devices

Create a base class SmartDevice with method activate() printing "Activating device...".
Subclass as SmartLight and SmartSpeaker.
SmartLight.activate() → "Activating light..."
SmartSpeaker.activate() → "Activating speaker..."
Loop over both and call .activate().

Mini Task 2: Super-powered Appliances
Create a base class Appliance with method start() printing "Starting appliance...".
Subclass WashingMachine and SmartWashingMachine.
WashingMachine.start() should call super().start() then print "Washing clothes...".
SmartWashingMachine.start() should call super().start() then print "Optimizing wash cycle...".

Mini Task 3: 3-Level Inheritance
Animal → method speak() → "Animal sound".
Dog(Animal) → super().speak() then "Bark".
SmartDog(Dog) → super().speak() then "Translating barks to human language".
Create a SmartDog and call .speak().
